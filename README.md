# 🎨 AI Image Editor

A modern and interactive Image Processing Web Application built using Python, Streamlit, and OpenCV.

This project allows users to upload an image, apply multiple image filters in real-time, preview changes instantly, and download the edited image.

---

## 🚀 Features

✅ Blur Filter
✅ Sharpness Control
✅ Brightness Adjustment
✅ Contrast Adjustment
✅ Grayscale Mode
✅ Edge Detection
✅ Download Edited Image
✅ Modern UI Design using Streamlit

---

## 🛠 Technologies Used

* Python
* Streamlit
* OpenCV
* NumPy
* Pillow
